# customer-case-analysis-python-sql
A business analytics portfolio project using Python, SQL, SQLite, and fictional customer case data to track cases, analyze performance, and generate reports. 

# Customer Case Tracking and Analysis System

## Project objective

This business analytics portfolio project uses Python, SQL, SQLite, and CSV files to analyze fictional customer-service cases.

The project demonstrates:

- Creating and connecting to a SQLite database
- Storing fictional case and specialist records
- Filtering and sorting cases with SQL
- Using SQL joins and aggregate functions
- Identifying open, overdue, and high-priority cases
- Calculating case counts and average resolution time
- Exporting analysis results to CSV with Python

## Technologies

- Python
- SQL
- SQLite
- CSV
- GitHub

## Data notice

All customer names, specialists, and case information used in this project are fictional. No real customer or employer information is included.

## Planned reports

1. Open cases
2. Overdue cases
3. High-priority cases
4. Cases grouped by status
5. Cases grouped by assigned specialist
6. Average resolution time
7. Exported CSV summary report
